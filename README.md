# 佐藤医院 Webサイト

制作依頼書 `SATO-WEB-2025-001`、掲載原稿 `SATO-WEB-2025-002`、レビュー資料を基準に制作した6ページ構成の静的サイトです。

## ページ

- `index.html`：トップページ
- `news.html`：お知らせ一覧・詳細
- `medical.html`：診療内容
- `about.html`：当院について
- `access.html`：診療時間・所在地
- `faq.html`：よくあるご質問

## お知らせの更新

お知らせは `js/news-data.js` の `window.SATO_NEWS` で一元管理しています。ここで記事を追加・編集すると、トップページの最新5件、お知らせ一覧、詳細ページへ反映されます。

記事には次の項目を設定します。

- `id`：URLに使用する重複しない半角英数字とハイフン
- `date`：`YYYY-MM-DD` 形式の公開日
- `title`：記事タイトル
- `body`：段落ごとの本文配列

新しい記事は配列の先頭へ追加します。公開前に、公開日・本文の承認記録を `docs/review/content-checklist.md` へ残してください。

## 公開前確認

`docs/review/content-checklist.md` に従い、制作依頼書・掲載原稿との一致、レスポンシブ表示、キーボード操作、画像表示を確認します。原稿と異なる文言を使う場合は、同チェックリストの承認記録を先に更新します。
